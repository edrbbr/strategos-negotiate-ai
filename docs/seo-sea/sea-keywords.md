# SEA — Keyword-Listen (Doku)

Match-Type-Konvention:
- `keyword` = Broad
- `"keyword"` = Phrase
- `[keyword]` = Exact

Initial empfohlen: **nur Phrase + Exact**. Broad erst, wenn Negativlisten reif sind.

## Match-Strategie über Zeit

| Woche | Strategie |
|---|---|
| 1–2 | Nur Exact, Conversion-Volumen messen |
| 3–4 | Phrase aktivieren, Suchanfragen-Bericht täglich prüfen, Negativliste verdichten |
| 5–8 | Phrase erweitern, gut performende Exact-Variationen hochbieten |
| 9+ | Broad selektiv (max. 1 Keyword pro AG), Smart-Bidding evaluieren |
